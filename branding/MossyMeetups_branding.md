# Mossy Meetups Branding (Concept C)

Brand direction
- Name: Mossy Meetups
- Vibe: mossy, earthy outdoors + Dead-head vibe
- Core promise: easy planning, clear schedules, simple invites, campfire magic

Mood & audience
- Music-loving families and friends; teens included
- Tone: friendly, nostalgic, confident, a touch whimsical

Color palette (pinecone motif)
- Base/Background: #F3F6F1
- Surface: #FFFFFF
- Text: #1A1A18
- Pinecone Brown: #5B3A2E
- Forest Green: #2E7D32
- Moss Green: #6A9A4F
- CTA: #D97706 / #E4A11B

Typography
- Display: Fraunces or Playfair Display
- Body: Inter or Source Sans Pro
- Scale: base 1rem; headings +1.25-1.5x

Logo concepts (C)
- Pinecone + waveform integrated
- Pinecone with musical notes woven into the pattern
- Clean wordmark with pinecone glyph

Tagline (optional)
- Plan together. Camp together. Mossy vibes.

UI/UX direction
- MVP: Week view + event list
- Private Groups, Event creation, RSVP, calendar export, email + in-app notif
- Auth: magic link; no anonymous guests

MVP data model
- User, Group, Event, DateOption, RSVP, Invite

Tech stack
- Frontend: Next.js + Tailwind
- Auth: NextAuth (magic link)
- Backend: Prisma + PostgreSQL
- Deployment: Dokploy on VPS
